OMORI em português

O que é

Feito pela equipe OMORI Brasil, o mod traduz todo o jogo OMORI para português por inteiro, desde falas até sprites, em todas as rotas. Nosso objetivo é providenciar uma maior acessibilidade para os brasileiros conseguirem usufruir a obra totalmente.

Trabalhamos duro para fazer algo fiel a língua original. Espero que gostem!

A tradução é independente, feita por fãs do jogo. Não temos intuito lucrativo e não há relação alguma com OMOCAT, LLC.

Passos de instalação

  1. Abra o arquivo zip do mod, dentro dela, selecione uma pasta chamada www.
  2. Após selecionar a pasta, coloque em “extrair para”.
  3. O local que você precisa extrair a pasta é na pasta geral do jogo, a pasta OMORI, ela normalmente fica em Arquivos de programas(x86) - Steam - steamapps - common - OMORI(extraia a pasta aí).
  4. O Winrar irá perguntar se você gostaria de substituir arquivos, clique em “Sim”.
  5. Agora é só iniciar o jogo pelo o botão “iniciar” da Steam, se a tela de aviso do jogo estiver traduzida, significa que a tradução foi aplicada com sucesso.

É uma boa ideia guardar seus saves caso haja algum problema na instalação que seja preciso a desinstalação do jogo. Eles ficam na pasta “save” dentro da “www”.
